# 匿名文件传输网站

一个简洁的匿名文件传输网站：
- 上传文件后生成 8 位提取码。
- 其他设备打开网站输入提取码即可下载。
- 支持大文件分片上传、断点续传、上传/下载进度条。
- 新增欢迎页 + 左上角小登录按钮，输入口令 `Welcome!2026` 才可进入上传下载页。
- 登录成功后会写入浏览器 cookie（7 天），同一设备下再次访问可免登录。
- 文件首次下载完成后自动从服务器删除，再次下载会提示文件不存在。

## 快速开始

```bash
npm install
npm start
```

然后访问 `http://localhost:3000`。

## 功能说明

- **分片上传**：前端将文件切分为 5MB 分片逐片上传。
- **断点续传**：上传中断后，再次选择同一文件会从已上传分片继续。
- **提取码下载**：通过提取码查询文件信息并下载。
- **一次性下载**：文件被下载一次后会立即从服务器删除。
- **大文件下载支持**：后端支持 HTTP Range，可断点下载。

## 目录结构

- `server.js`：后端 API 与静态服务。
- `public/`：前端页面与脚本。
- `storage/`：上传索引、临时分片与最终文件。
